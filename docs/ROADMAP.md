# 路线图

## MVP 屏幕清单

以下是游戏的主要屏幕/功能模块：

1. **开局** - 游戏开始界面（已落地）
2. **主界面** - 游戏主菜单（已落地）
3. **修炼** - 修炼界面（已落地）
4. **探索** - 探索界面（已落地）
5. **探索深入** - 事件 JSON 系统（已落地）
6. **收手** - 停止探索
7. **炼丹** - 炼丹系统（已落地）
8. **突破准备** - 突破境界前的准备（已落地）
9. **突破结算** - 突破结果展示（已落地，含保底机制）
10. **死亡结局** - 角色死亡处理（已落地）
11. **本局总结** - 单局游戏总结（已落地）
12. **图鉴** - 游戏内容图鉴（含事件图鉴）
13. **设置** - 游戏设置（已落地）
14. **存档** - 存档/读档功能（已落地）
15. **传承** - 传承系统
16. **商店** - 商店系统
17. **背包** - 物品管理
18. **任务** - 任务系统
19. **成就** - 成就系统
20. **统计** - 游戏统计数据
21. **奇遇回顾** - 探索事件回顾（占位）
22. **丹药配方系统** - 丹药合成与配方（占位）
23. **装备/功法** - 装备与功法系统（占位）
24. **事件图鉴** - 探索事件图鉴（占位）
